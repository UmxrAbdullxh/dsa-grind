# LeetCode Grind

Personal repository of LeetCode problem solutions, organized by problem number.

## Structure
- Each solution is a standalone file named `<problem-number>.<problem-slug>.<ext>`.
- Problems are solved in **Python** (`.py`) and **TypeScript** (`.ts`).
- Files can be executed directly (no build step required for Python; TypeScript files may need `ts-node` or similar).

## Running Solutions

**Python**
```bash
python 0001.two-sum.py
```

**TypeScript**
```bash
npx ts-node 0003-longest-substring-without-repeating-characters.ts
```

## Notes
- Solutions are self‑contained; each file includes the class/function definition and any necessary imports.
- The repository is a work‑in‑progress; problems are added as they are solved.
